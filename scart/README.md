# Overview
This is the front end of the inventory manageer that consumes below APIs -

product-create -- To add a new item to the inventory
product-list -- To fetch the existing items
product-delete -- To delete an item
product-update -- To updates an items
product-filter -- To filter list data based on min and max price bracket (API is there but not implemented on front end)

Features -
1. Validation using Formik and Yup
2. API calls using axios
3. Chakra for UI components
